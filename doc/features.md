# Feature List

## Supported version

RIFT [RFC 9692](https://www.rfc-editor.org/rfc/rfc9692.pdf)

Thrift data model version: 6.1 (w/o auto-evpn or auto-fabric support)

| Adjacencies using IPv6 Link and Multicast Addresses | Yes |
